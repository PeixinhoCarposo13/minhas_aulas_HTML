Vamos aprender HTML

-> É o esqueleto do site.
-> Extensão "Live server", servidor local simples para o meu site.
-> Nome do arquivo deve ser "index.html" por questão de padrão.
-> Para rodar clique com o botao esquerdo e selecione "Open with Live server".
-> Para fazer uma tag de titulo digite "h1"
-> Temos uma ordem do "h1" até o "h6" segundo assim do Titulo Maior até o menor 
dos subtitulos.
-> A primeira coisa que devemos escrever no arquivo é <!DOCTYPE html> pois ele 
informa a versão do nosso HTML.
-> A segunda coisa é colocar a tag "html" pois tudo o que for ser feito no arquivo
vai ter que estar entre essa tag.
-> Terceiro passo é separar as informações que vão aparecer visivelmente no site das 
que não vão aparecer visivelmente no site.
-> Tag "head" para elementos não visuais.
-> Tag "Body" para elementos visuais.
-> Tag "title" para o titulo na "head", entao nao vai ser o titulo dentro do site.
-> Tag "meta" para inserir metadados no "head", sendo esses dados escritos dentro dos
"<>" ao lado da escrita "meta", voce deve deixar claro o tipo do dado ( por exemplo se 
vai ser uma descrição) e o conteudo desse dado ( se for uma descrição escreva essa descrição).
-> No "body" para escrever o cabeçario use "header".
-> No "body" use para escrever o corpo principal "main".
-> No "body" use "footer" para escrever o rodapé.
-> Use a tag "p" para paragrafo.
-> Use a tag "section para dividir seu conteudo em seções.
-> Use a tag "strong" para deixar uma frase em destaque.
-> Use a tag "em" para deixar em um destaque secundário.
->Use a tag "b" para deixar em negrito e "i" para deixar em itálico.
->Use a tag "br" para quebrar linha.
-> Use a tag "abbr" para mostrar o significado de uma abreviação.
-> Use a tag "form" para o imput do usuário ficar organisado da forma correta.
-> Use a tag "input" com "radio" para receber os dados do usuário em multipla escolha, 
{ input type="radio" value="pyton" id="p1a" name="pergunta1" } deve ter esses dados.
-> Use a tag "label" para adicionar a escrita que vai aparecer na tela, 
{ <label for="p1a"> PYTHON </label> }.
-> Use a tag "input" com "text" para receber os dados do usuário em texto, 
{ <input type="text" id="p2" name="pergunta2" placeholder="Digite aqui seu texto"> } deve ter esses dados.
-> Use a tag "input" com "password" para receber os dados do usuário como senha,
{ <input type="password" id="p3" name="pergunta3" placeholder="Escreva sua senha"> }
-> Use a tag "input" com "data" para receber os dados do usuário como data,
{  <input type="date" id="p4" name="pergunta4"> }
-> Use a tag "input" com "data" para receber os dados do usuário como data,
{ <input type="date" id="p4" name="pergunta4"> }
-> Use a tag "input" com "checkbox" para receber os dados do usuário como multipla escolha em que 
pode ser selecionado mais de uma opção, 
<input type="checkbox" id="p5a" name="pergunta5" value="Javasscript">
-> Use a tag "label" para adicionar a escrita que vai aparecer na tela,
<label for="p5a">Javasscript</label>
-> Use a tag "input" com "file" para receber os dados do usuário como arquivo,
 <input type="file" id="p6" name="pergunta 6" >
 -> Use a tag "select" para fazer um menu suspenso.
 -> Use a tag "option" papa preencher o select.
 -> Use a tag "img" para adicionar uma imagem.
 -> Use a tag "table" para criar uma tabela.
 -> Use a tag "tr" para fazer uma linha de uma tabela.
 -> Use "td" para escrever dentro do "tr".
 ->Use a tag "thead" para criar o cabeçalho da "table"
 ->Use "th" para adicionar escrita na "thead"
 ->Use "tbody" para adicionar as escritas do corpo da tabela
 ->Use "tfoot" para escrever o rodapé da tabela.
